# 相关说明

## 一、使用说明：

> 可能已经失效了……

### 1. 夸克网盘：

1. 添加 Synoogy 协议；
2. 地址填写 quark.example.com；
3. 密码填写 Cookie，登录网页版抓包获得。

### 2. PikPak：

1. 添加 Synoogy 协议；
2. 地址填写 pikpak.example.com；
3. 直接填你的 PikPak 账号密码即可。
